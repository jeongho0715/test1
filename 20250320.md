



## Relu
![image](/image/graph(relu).png)

### Concept

$y=max(0, x) \tag{1}$

Equation above can be denoted by

$y=\begin{cases}0,\ x> 0\\x,\ x\leq 0\end{cases}$

### Naive Method
```python
def naive_relu(x):
    assert len(x.shape) == 2
    x = x.copy()
    for i in range(x.shape[0]):
        for j in range(x.shape[1]):
            x[i, j] = max(x[i, j], 0)
    return x
```
